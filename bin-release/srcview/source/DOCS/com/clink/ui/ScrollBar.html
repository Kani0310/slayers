Classpublic class ScrollBar
InheritanceScrollBar flash.display.Sprite

This class creates a scrollbar component with arrow buttons. This draws the scrollbar programatically instead of importing movieClips. The static functions can be used to change the color of the different scrollBar parts.

public static function setScrollBarColors(buttonColor:String = #aaaaaa, trackColor:String = #ffffff, buttonArrowColor:String = #333333, handleColor:String = #aaaaaa):void

This method can set/change all scrollBar colors of all instantiated scrollBars

Parameters

buttonColor:String (default = #aaaaaa) — Arrow Button color can be passed in as a css value ex. #FFFFFF or as an actionscript hex value 0xFFFFFF, either will work
 
trackColor:String (default = #ffffff) — Track color can be passed in as a css value ex. #FFFFFF or as an actionscript hex value 0xFFFFFF, either will work
 
buttonArrowColor:String (default = #333333) — Arrow color on the buttons can be passed in as a css value ex. #FFFFFF or as an actionscript hex value 0xFFFFFF, either will work
 
handleColor:String (default = #aaaaaa) — handle color can be passed in as a css value ex. #FFFFFF or as an actionscript hex value 0xFFFFFF, either will work
