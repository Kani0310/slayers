Classpublic class LayoutBox
InheritanceLayoutBox flash.display.Sprite

The LayoutBox class is a multipurpose tool used to laying out objects on the screen. The layout box is placed onto the stage and objects are added to it's displaylist using the addChild() method. When the LayoutBox is instantiated there are optional parameters which can be set to determine if the objects will be positioned horizontally, vertically or on a grid. There are also padding properties used for setting the space between objects. The colPadding option is used when grid mode is enabled.
