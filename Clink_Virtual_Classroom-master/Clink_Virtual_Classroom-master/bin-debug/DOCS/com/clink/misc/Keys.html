Classpublic class Keys
InheritanceKeys Object

This class is used for detecting key presses. The public function init must be executed first with a stage reference. Than the public static var keys can be accessed where keys is an array and keys[keyCode] will return a true or false boolean value which can be used in a conditional. Example usage: if(Keys.keys[Keyboard.UP]){do this}
